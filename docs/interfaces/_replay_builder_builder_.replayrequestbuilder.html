Interface ReplayRequestBuilder

Builds ReplayRequestParam.

You must provide builder with

  • Exchanges and its channels to filter-in by calling functions with exchange name or by calling exchange function to select arbitrary exchanges and channels.
  • Time range by functions start and end or range to set them both at the time.
  • What data format to fetch.

Get ReplayRequestParam by specifing the data format either by calling

  • {@link asRaw()} to get results in a raw format which format is specific to each exchange
  • {@link asCSVLike()} to get results in a csv-like formatted version

  • end(date: AnyDateInstance): ReplayRequestBuilder
    • Defined in replay/builder/builder.ts:76

    Set the end date and time of filtering.

    If only date was given, it will be comprehended as the very end of the day.

    Example: 2020-03-20 is the same as 2020-03-21 00:00:00

    Beware that the end date time is exclusive, means 2020-03-20 08:17:00 is not included but 2020-03-20 08:16:59.999999999 is.
